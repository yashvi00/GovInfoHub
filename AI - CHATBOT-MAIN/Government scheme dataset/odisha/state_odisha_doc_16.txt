
Odisha Kalia Yojana 3rd Phase List | Odisha Kalia Yojana 3rd Phase Beneficiary List | Kalia Yojana Odisha Name List Phase 3 | Odisha Kalia Yojana Phase 3 Green List | Odisha Kalia Yojana 3rd List 2019 | Kalia Yojana Odisha Name List | Kalia Yojana Third beneficiary list | 3rd Phase Kalia Scheme List of Beneficiaries 
Odisha government has now released the 3rd Phase Draft Beneficiaries List (DBL) or green list under Krushak Assistance for Livelihood and Income Augmentation (KALIA) Scheme 2019. The complete name wise Kalia Yojana Phase 3 beneficiary list is available online to download at the official website kalia.co.in. Those farmers who have previously filled Part A green form for inclusion of name in Kalia Scheme Phase 3 can now check their name in the Kalia Yojana Third Phase Green List of beneficiaries.
Kalia Yojana Thirs List of Beneficiaries (Kalia Scheme 3rd Phase Green List) is released online at the official Kalia Portal. 3rd Phase Kalia Yojana Odisha Name List is available for downloading purposes in PDF format by just selecting their district name and gram panchayat (block).

In the new Kalia Scheme list of beneficiaries, the state govt. has included all the eligible farmers who were left out in the 1st phase Kalia Yojana 2019 beneficiary list & 2nd Phase Kalia Scheme Beneficiaries List and have also filled green form for inclusion of name in Kalia Yojana 2019.
Odisha Kalia Yojana 3rd Phase Green List of Beneficiaries 2019
Here is the complete process to download Kalia Yojana Phase 3 list of beneficiaries or find name online in Kalia Scheme 3rd phase beneficiary list:-

Firstly visit the official KALIA Scheme website at kalia.co.in
At the homepage, click at the “ହିତାଧିକାରୀ (Beneficiary List)” option present in the header (main menu).
Direct Link – http://green.kalia.co.in/BeneficiaryGreen.aspx
Accordingly, Kalia Yojana 3rd Phase Beneficiary Green List will appear as follows:-
Kalia Yojana 3rd Phase List Beneficiaries
In this Kalia Scheme Green List of Beneficiaries for third phase, small and marginal farmers along with landless agricultural labourers can select the name of district and block / urban local bodies (ULBs) and then click at the “View” option.
Afterwards, the district wise or block wise or gram panchayat wise list having PDF files which contains the name of eligible farmers will be available to download online. This Kalia Yojana 3rd phase beneficiary list is both for small and marginal farmers as well as landless agricultural labourers as shown below:-
Odisha Kalia Yojana 3rd Phase Beneficiary List District Wise
Here all the beneficiaries can click at the PDF files shown under ‘Download option‘ to download the 3rd Phase Kalia Yojana Green List 2019. As a further step, Kalia Scheme Proceed to “PDF View option” will appear where beneficiaries can enter captcha as shown below:-
Odisha Kalia Scheme Third Phase PDF List
Afterwards, the Kalia Yojana 3rd Phase Beneficiary List will appear as shown below:-
Kalia Yojana 3rd Phase List
This 3rd Phase Kalia Scheme Green list of beneficiaries contains the name of village, name of farmers, husband / father name and gender along with the district, block and gram panchayat names.

For more details on Kalia Yojana, visit the official website http://kalia.co.in/index1.html
